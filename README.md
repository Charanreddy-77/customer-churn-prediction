# Customer Churn Prediction

This project predicts whether a customer will churn or not using Machine Learning.

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Dataset
Telco Customer Churn Dataset

## Features
- Senior Citizen
- Tenure
- Monthly Charges
- Total Charges

## How to Run
pip install flask pandas scikit-learn joblib
python train_model.py
python app.py
